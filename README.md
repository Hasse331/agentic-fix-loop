# FixLoop

FixLoop turns user-reported issues into agent-ready fixes.

## What You Can Use Right Now

- `@fixloop/core`: issue validation and agent-facing shape
- `@fixloop/server`: `POST /reports` and `GET /issues`
- `@fixloop/widget`: browser payload helpers and a working vanilla JS modal
- `@fixloop/storage-supabase`: Supabase adapter and starter schema
